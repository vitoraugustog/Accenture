Ambientação Windows

1. Baixar o Kit de Desenvolvimento Java (JDK) https://www.oracle.com/br/java/technologies/javase-downloads.html
2. Instalar o Kit de Desenvolvimento Java (JDK)
3. Após a instalação, verifique se as pastas da JDK e da JRE estão devidamente instaladas. Elas se encontram no diretório "Arquivos de Programas" (dentro do diretório C). O caminho do diretório é algo parecido com :"C:\Program Files\Java".
4. Baixar o Eclipse IDE https://www.eclipse.org/downloads/
5. Instalar o Eclipse IDE será aberta a janela de instalação da aplicação, selecionar a opção "Eclipse IDE for Java Developers". Após isso é só dar next e finalizar.
6. Antes do Eclipse IDE ser inicializado, aparecerá uma janela solicitando que você escolha pasta onde os projetos Java serão armazenados. Deixe o diretório padrão na pasta de usuário e clique no botão Launch.

Preparação do projeto

1. Baixar os arquivos que estão no zip
2. Abrir o Eclipse IDE 
3. Será inicializado e será apresentada a tela de boas vindas. Clique no símbolo de “x” (fechar) da aba Welcome, no canto superior esquerdo da janela, para fechar a tela de boas vindas.
4. Clique em File > Import > Maven > Existing Maven Projects > Escolha a pasta de destino onde o projeto foi baixado e finalize a exportação.
5. Dentro do arquivo mainTest.java alterar o destino do driver localizado na linha 13, para o caminho do computador que irá rodar o teste
6. Clique no botão verde com o botão do play branco que o teste será executado.
